# Car Rental Project

Учебный Java Web-проект для аренды автомобилей.  
Приложение написано на **Java Servlets + JSP** с использованием **JDBC** и **PostgreSQL**.

## Описание проекта

Car Rental Project — это простое веб-приложение, в котором пользователь может зарегистрироваться, войти в аккаунт, посмотреть список автомобилей и управлять статусом машины.

Проект создан как учебный pet-project для практики:
- Java Core
- Jakarta Servlets
- JSP / JSTL
- JDBC
- DAO-слоя
- DTO, Entity, Mapper
- валидации данных
- работы с PostgreSQL
- разделения приложения на слои

## Основная функциональность
- Регистрация клиента
- Авторизация клиента
- Просмотр списка автомобилей
- Просмотр детальной информации об автомобиле
- Добавление нового автомобиля
- Бронирование автомобиля
- Возврат автомобиля
- Отметка автомобиля как сломанного
- Ремонт автомобиля
- Валидация данных при создании клиента и автомобиля
- Работа с базой данных через DAO

## Статусы автомобиля

В проекте используются 3 статуса автомобиля:

1. `AVAILABLE` - Машина доступна для бронирования 
2. `RENTED` - Машина забронирована 
3. `BROKEN` - Машина сломана

## Архитектура проекта
Проект разделён на несколько слоев:
src/main/java/org/example/car/rental/project

dao          # Работа с базой данных
dto          # Объекты для передачи данных
entity       # Основные сущности проекта
exception    # Пользовательские исключения
mapper       # Преобразование DTO в Entity и обратно
service      # Бизнес-логика приложения
servlets     # Servlet-контроллеры
util         # Утилитные классы
validator    # Валидация входных данных


JSP-страницы находятся в: 
src/main/webapp/WEB-INF/jsp


## Сущности

### Car
Автомобиль содержит:
- id;
- brand;
- model;
- productionYear;
- pricePerDay;
- carNumber;
- status.

### Client
Клиент содержит:
- id;
- firstName;
- lastName;
- email;
- password;
- phone;
- passportNumber;
- createdAt.

### CarStatus
Enum со статусами автомобиля:
AVAILABLE, RENTED, BROKEN

## Основные страницы и маршруты
| URL | Метод | Описание |
|---|---:|---|
| `/registration` | GET / POST | Регистрация клиента |
| `/login` | GET / POST | Авторизация клиента |
| `/cars` | GET | Список автомобилей |
| `/cars/info?carId={id}` | GET | Информация об автомобиле |
| `/car-create` | GET / POST | Добавление автомобиля |
| `/cars/book` | POST | Бронирование автомобиля |
| `/cars/unbook` | POST | Возврат автомобиля |
| `/cars/broke` | POST | Отметить машину как сломанную |
| `/cars/fix` | POST | Починить машину |
